# Handwritten Notes Digitizer

Convert handwritten notes to digital text using Google Gemini AI Vision. Upload a photo of any handwritten document and get clean, editable digital text back instantly.

## Overview

This project uses Google Gemini multimodal vision capabilities to perform OCR on handwritten notes, making it easy to digitize lecture notes, sketches, and handwritten documents.

## Features

- Upload handwritten note images (JPG, PNG)
- Extract and structure text using Gemini Vision API
- Clean, formatted digital output
- Supports various handwriting styles

## Tech Stack

| Tool | Purpose |
|------|---------|
| Python | Core language |
| Google Gemini AI | Vision and text extraction |

## Getting Started

1. Clone the repository and install dependencies:
   pip install google-generativeai pillow

2. Set your Gemini API key:
   export GEMINI_API_KEY=your_api_key_here
